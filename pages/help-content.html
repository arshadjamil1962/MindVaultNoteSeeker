Mind Vault - Note Seeker your personal assistant chrome extension for securely storing, organizing, and finding your thoughts, knowledge, and resources, all in one place.


Mind Vault Notes Registeration

Imagine you're browsing your favorite website and come across a brilliant idea or important information.


With Mind Vault Note Seeker, capturing that is as simple as

highlighting the text, right-clicking, and selecting the "Mind Vault Note Addition" option.

  • an Entry screen opens in a new tab, carrying forward the selected text, the source URL, and the current date automatically.
  • Your job? Just fill out the required details like the Title, Priority, Tags, and Keywords.
  • Tags, and Keywords are important in categorization of content in order to facilitate search.
  • Filling out Tags, and Keywords is very simple, just select from the list or type new one.
  • Selected once are displayed as button element with description and a "x" mark.
  • To remove any element from selection click on "x" mark.
  • Want to link current note content with another existing notes? Can add a cross-reference too—it’s optional but super handy!
  • Once Fill out information and satisfied, just click **Save**, and your note is securely stored in the database.
  • The **Entry screen** tab is closed.

Great! You successfully added your note in your Notes Collection.


Buttons Displayed at Extension Menu

Launching the MindVaultNoteSeeker extension, you will be presented with the following options:

  • Help: Opens this help content to guide you through the extension's usage.
  • Note Seeker - Mind Valut: Provides an interface Screen to View Notes Collection.

Mind Vault Notes Collection View

Your notes are displayed in a neat grid of four(4) per page.

  • Delete Button next to each note, allow you to remove that note from the collection on confirmation.
  • A dialog box appears with Notes Information, confirming the Delete Request.
  • On Confirmation said Note be deleted from the Collection.

Navigate through your collection effortlessly using the pagination controls at the bottom of screen.

  • First: Move to the First Page
  • Previous: Move to Previous Page
  • Next: Move to Next Page
  • Last: Move to Last Page.

Note: You also find info. about total number of records and the specific notes currently visible.


Mind Vault Notes Detailed View

The details about any note can easily be seen by clicking on the note you want to visualize.

  • A dialog box style window pops up, displaying details about the Note.
  • Cross references Notes attached will be displayed as accordions.
  • click on any of the accordion, opens the accordion to show respective information.

Note: You can even navigate straight to the original source with a single click on source link.


Mind Vault Notes Filters

Want to find something specific? No problem.

On the left side of the screen, you’ll discover powerful filters Or Search Options by title, abstract, dates, tags, keywords, and even priority. This makes narrowing down your search a breeze.


Fill out desired filter with your search entry and click "Apply Filters", results Notes listing based on filter options, navigate them, check details.

  • Filtering the search you can use individual fields or any combination instead.
  • Only filter on "Title", "Abstract", "Dates", "Tags", "Keywords", and "priority. OR
  • filter on "Titles" includes ?search-text that has following "Tags"/ "Keywords" etc.
  • Update, remove and/or type new search entries and click "Apply Filter" again to continue.
  • Finally filters can be cleared at a glance by click "Clear All".

SQLite Database Requirement

SQLite's database file is included inside the source file structure, which does not require any database server to set up.

The Database file is named "MindVaultNoteSeeker.db" and its location is

"mindVaultNoteSeeker\DatabaseSQLite\Database"


Important to Note!

  • Tags and Keywords classifies the Note's content and has similar importance as "Title" that enhance searching.
  • Closing the browser without saving, in Entry Screen, results in a loss of current additional screen information. Rest of the data remains intact.
  • Currently SQLite database is set up locally, so it can not be accessed from another laptop or computer.
